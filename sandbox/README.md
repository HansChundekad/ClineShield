# Sandbox Directory

This directory is for testing ClineShield hooks safely.

## Purpose
All test files here can be modified/deleted without affecting real code.

## Test Files
- `test-files/sample.txt` - Plain text file for editing tests
- `test-files/data.json` - JSON file for parsing tests

## Instructions for Testing with Cline

Ask Cline to perform operations on files in this sandbox directory.
The hooks should capture all tool usage.
